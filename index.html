Inicio

Bienvenido a TECNOYAJ SOLUTIONS, tu socio en soluciones web. Descubre todo lo que podemos hacer por ti.

Proyecto: Sistema web para préstamos de recursos tecnológicos en la universidad.


Este sistema web permite la gestión de préstamos de recursos tecnológicos autorizados en la universidad.

Diseño de interfaces y Formularios

Diseñamos 4 formularios que le permiten a los usuarios registrar nuevas personas, agregar recursos tecnologicos, realizar prestamos y consultar el inventario de equipos disponibles.




Desarrollo base datos

Creamos una base datos con las tablas relacionales de cada formulario.




Recursos utilizados

Los recursos o aplicativos utilizados en el desarrollo de este proyecto fueron:

  • Motor de base de datos Mysql
  • Entorno de desarollo Apache Netbeans IDE 16
  • Editor de codigo fuente Visual estudio code
  • Paquete de software libre Xampp
